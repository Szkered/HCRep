# Hazelcast-CIBGrp5

## Very brief intro to maven

### 1. Basic survival skills in mavenland
Build:

```Bash
mvn clean package
```

Run: double click the jar file in target directory, or in command line

```Bash
java -jar $module-name/target/$module-name-1.0-SNAPSHOT.jar
```
Test:
```Bash
mvn test
```
